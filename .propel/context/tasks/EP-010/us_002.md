# US-002 — Funnel Visualisation with Stage Conversion and AI Accuracy Confusion Matrix

## User Story

**As a** recruiting manager,  
**I want** a funnel chart showing stage-by-stage candidate drop-off rates and an AI accuracy confusion matrix comparing AI recommendations against actual HR decisions,  
**So that** I can identify which stages lose the most candidates and measure how well the AI screening is performing.

## Business Value

Funnel visibility pinpoints exactly where candidates drop off, enabling targeted process improvements. The AI accuracy matrix quantifies whether the screening model is adding value, supporting the decision to keep, tune, or retrain it.

## Acceptance Criteria

### Scenario 1: Funnel shows correct counts and conversion rates at each stage
**Given** 100 applications exist with 40 shortlisted, 20 completed all interviews, and 5 received offers  
**When** the funnel chart renders  
**Then** four funnel bands display with correct counts and conversion rates: Applications (100), Shortlisted (40%, 40), Interviews Complete (50%, 20), Offer Extended (25%, 5).

### Scenario 2: Drop-off rate highlighted for the largest decline
**Given** the largest drop occurs between "Shortlisted" and "Interviews Complete"  
**When** the funnel renders  
**Then** that band transition is highlighted in amber with a tooltip showing the exact drop count and percentage.

### Scenario 3: AI confusion matrix shows correct quadrants
**Given** screening recommendations and HR decisions are available  
**When** the confusion matrix renders  
**Then** true positives, false positives, true negatives, and false negatives are correctly calculated and displayed in a 2×2 grid.

### Scenario 4: Matrix filters to a selected requisition
**Given** the manager selects a specific requisition  
**When** the filter is applied  
**Then** the confusion matrix recalculates using only applications from that requisition.
