# US-002 — Audit Log Viewer with Search, Filter, Pagination, and CSV Export

## User Story

**As a** compliance officer,  
**I want** to search and filter the audit log by actor, event type, entity, and date range, with CSV export completing in under 30 seconds,  
**So that** I can investigate specific incidents and produce audit reports for regulators without engineering support.

## Business Value

Self-service audit search eliminates engineering involvement in routine compliance requests, which can occur daily during audits. Sub-30-second CSV export ensures the tool is usable in time-sensitive regulatory contexts.

## Acceptance Criteria

### Scenario 1: Full-text search returns matching events
**Given** the compliance officer searches for actor email "john.doe@example.com"  
**When** the search executes  
**Then** all audit events where `actor_email = "john.doe@example.com"` are displayed, paginated at 50 per page.

### Scenario 2: Date range filter narrows results
**Given** the compliance officer applies a date range of "1 Jan 2026 – 31 Jan 2026"  
**When** the filter is applied  
**Then** only events with `created_at` within that range are shown.

### Scenario 3: CSV export completes in under 30 s
**Given** the current filter returns 50 000 audit events  
**When** the compliance officer clicks "Export to CSV"  
**Then** a CSV download begins within 30 seconds containing all matched events with all audit field columns.

### Scenario 4: Pagination navigates through large result sets
**Given** 5 000 events match the current filter  
**When** the compliance officer navigates to page 3  
**Then** events 101–150 are displayed and the URL reflects `?page=3`.
